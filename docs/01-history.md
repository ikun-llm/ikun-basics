# 第一章：大模型发展史

> AI 怎么从"人工智障"变成"鸡你太美"

---

## 一句话版本

AI 经历了"规则时代 → 机器学习时代 → 深度学习时代 → 大模型时代"四个阶段，从需要人写规则，到自己从数据中学规则，到现在能自己写诗、画画、聊天。

---

## 时间线（看这个就够了）

```
1950s  "AI"这个词被发明
  │     当时的人觉得10年就能造出超级智能（太天真了）
  │
1960s  感知机（Perceptron）
  │     最简单的"神经网络"，只有一层，能做简单分类
  │     相当于：ikun 刚开始练习，只会站着不动
  │
1970-80s  AI 寒冬 ❄️
  │     大家发现 AI 没想象中好用，资金断了，研究停了
  │     相当于：练习生海选被淘汰，回家反思
  │
1990s  机器学习兴起
  │     不再手写规则，让机器从数据中"学习"
  │     代表：SVM（支持向量机）、决策树、随机森林
  │     相当于：练习生开始跟着老师学基本功
  │
2012   深度学习爆发 🔥 — AlexNet
  │     用 GPU 训练深层神经网络，图像识别准确率暴涨
  │     相当于：练习生突然开窍了，学会了第一个才艺
  │
2014   GAN（生成对抗网络）
  │     两个网络互相对抗，一个造假一个辨真
  │     能生成人脸、风景等图片
  │
2015   ResNet（残差网络）
  │     解决了"网络太深训不动"的问题，152层都能训
  │
2017   ⭐ Transformer 诞生 ⭐
  │     Google 的论文《Attention Is All You Need》
  │     用"注意力机制"替代了 RNN，从此改变了一切
  │     相当于：练习两年半的转折点！
  │
2018   BERT / GPT-1
  │     BERT：读懂文章（完形填空式训练）
  │     GPT-1：学会写作（预测下一个词）
  │
2019   GPT-2
  │     7.5亿参数，写出的文章已经很像人了
  │     OpenAI 一开始不敢公开，怕被滥用
  │
2020   GPT-3
  │     1750亿参数，"大力出奇迹"的开始
  │     发现：模型够大 + 数据够多 = 涌现能力
  │
2021   DALL-E / Diffusion Models
  │     AI 开始能画画了（文字→图片）
  │
2022   ChatGPT 🚀
  │     GPT-3.5 + RLHF（人类反馈强化学习）
  │     全球爆火，AI 走进普通人生活
  │     相当于：练习生终于出道了！
  │
2023   GPT-4 / LLaMA / Qwen
  │     多模态（能看图+聊天）
  │     开源大模型崛起
  │
2024   Sora / Claude 3 / DeepSeek
  │     AI 视频生成 / 更强的推理能力
  │     DeepSeek 用 GRPO 训练推理模型
  │
2025   DeepSeek-R1 / ikun-2.5B
  │     推理模型时代
  │     以及...练习时长两年半的 AI 练习生诞生！🐔
  ↓
```

## 几个关键转折点

### 转折 1：从规则到学习

```
以前（规则时代）：
  程序员手写：如果邮件包含"中奖"→ 垃圾邮件
  问题：规则写不完，新花样层出不穷

现在（学习时代）：
  给机器 10 万封邮件（标注好哪些是垃圾）
  机器自己学会判断
  优点：能应对没见过的新花样
```

### 转折 2：从浅层到深层

```
浅层网络：2-3 层，能学简单模式
深层网络：几十上百层，能学复杂模式

为什么之前不用深层？
→ 训练不动（梯度消失问题）
→ 太慢（没有 GPU 加速）

2012 年之后：
→ GPU 便宜了
→ 数据多了（互联网）
→ 算法改进了（ReLU、BatchNorm、ResNet）
→ 深度学习起飞！
```

### 转折 3：从专用到通用

```
以前：一个模型只做一件事
  - 图像分类模型只能分类
  - 翻译模型只能翻译
  - 情感分析模型只能分析情感

现在（大模型时代）：一个模型什么都能做
  - ChatGPT 能聊天、写代码、翻译、总结、推理...
  - 为什么？因为"语言"是一切的基础
  - 学会了语言，就学会了知识
```

## 模型大小的演变

```
感知机 (1960s)     →  几个参数
LeNet (1998)       →  6 万参数
AlexNet (2012)     →  6000 万参数
ResNet (2015)      →  6000 万参数
BERT (2018)        →  3.4 亿参数
GPT-2 (2019)       →  15 亿参数
GPT-3 (2020)       →  1750 亿参数
GPT-4 (2023)       →  据传万亿级别
ikun-2.5B (2025)   →  2600 万参数（练习时长两年半）
```

> 你可能注意到了，ikun-2.5B 只有 2600 万参数，比 2012 年的 AlexNet 还小。但没关系，小而精也是一种美——鸡你太美 🐔

## 本章小结

| 时代 | 年代 | 核心思想 | 比喻 |
|------|------|---------|------|
| 规则时代 | 1950-1990 | 人写规则给机器 | 教练一个动作一个动作教 |
| 机器学习 | 1990-2012 | 从数据中学规则 | 看录像自学 |
| 深度学习 | 2012-2020 | 更深的网络+更多数据 | 进入专业训练营 |
| 大模型时代 | 2020-now | 大力出奇迹+对齐 | 出道成为偶像练习生 |
